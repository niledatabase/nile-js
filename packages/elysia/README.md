# @niledatabase/elysia

Seamless integration between [Nile Database](https://www.thenile.dev/docs/) and [ElysiaJS](https://elysiajs.com).

This package provides an extension that automatically bridges Nile's server-side capabilities with your Elysia application, handling request context propagation and route registration.

[See the docs for more details](https://www.thenile.dev/docs/auth/frameworks/elysia)
